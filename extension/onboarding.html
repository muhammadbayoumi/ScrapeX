Welcome to ScrapeX

A scraping tool that runs entirely on your machine. The extension is the interface — the engine is a small local app you install once.

Checking for the ScrapeX engine…
1

Install Python 3.12+

Get it from python.org/downloads (on Windows, tick “Add Python to PATH”).

2

Install the ScrapeX engine

pip install scrapex

A one-click installer is coming; for now this pip command installs the engine.

3

Register the launcher once, ever

python -m scrapex.cli install-native-host --extension-id …

This teaches Chrome to start the engine itself. The command is already filled in with this extension's id — copy it exactly as shown. After this step, a terminal is never needed again.

4

Start the engine

Or from a terminal, if the launcher is not registered yet: python -m scrapex.cli ui

Engine URL: …

Engine connected

You're all set. Open ScrapeX to add your first site.

ScrapeX is decentralized: your links and data stay on your machine. Nothing is sent to any server.
