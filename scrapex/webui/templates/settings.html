Privacy and security What is stored, where, and what is never done
Network exposureThe engine listens on 127.0.0.1 only. It is not reachable from your network.
SecretsThe Apps Script token is stored in the database on this machine and never shown again — only its last four characters. Google sign-in is a token file under your home folder.
Scraped contentTreated as untrusted. Every value is escaped before it reaches a page, so a product name cannot carry script into this interface.
Access controlsNever bypassed. A source that needs a sign-in is marked as such and reported honestly instead of being worked around.
Remote codeNever executed. The extension runs only the code it ships with.
